# Документация AILLM

Эта директория содержит всю документацию проекта AILLM.

## Документация

### ARCHITECTURE.md
Описание архитектуры проекта, основных компонентов и их взаимодействия.

### PROJECT_ANALYSIS.md
Детальный анализ проекта, включая найденные проблемы и рекомендации.

### FRONTEND_ANALYSIS.md
Анализ frontend части проекта с выявленными проблемами и предложениями по улучшению.

### LOGGING.md
Документация по единой системе логирования проекта.

### VALIDATION.md
Документация по системе валидации проекта для поддержания порядка и качества.

---

## Быстрый старт

1. **Валидация**: Начните с [VALIDATION.md](VALIDATION.md) для понимания системы автоматической проверки проекта
2. **Архитектура**: Изучите [ARCHITECTURE.md](ARCHITECTURE.md) для понимания общей структуры проекта
3. **Анализ**: Посмотрите [PROJECT_ANALYSIS.md](PROJECT_ANALYSIS.md) для понимания текущего состояния проекта
4. **Frontend**: Ознакомьтесь с [FRONTEND_ANALYSIS.md](FRONTEND_ANALYSIS.md) для работы с frontend
5. **Логирование**: Прочитайте [LOGGING.md](LOGGING.md) для использования системы логирования
